# 데이터 타입:

    원시타입: String, Number, Boolean, Bight, undefined, System, null

        @메모리 주소와 값을 대응시켰을 때, 재할당 시, 다른공간에 저장된다. = "불변성"

    참조타입: 객체 타입과 유사하다. 

        @참조타입의 "가변성"이란, 기존 메모리에서 참조를 통한 새로운 메모리를 할당하는 속성이다. 
        기존 메모리가 "불변성"을 가지는 것은 동일하다.
